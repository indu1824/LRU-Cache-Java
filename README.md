# LRU-Cache-Java
# 🚀 LRU Cache Implementation in Java

## 📌 Overview
This project demonstrates the implementation of a Least Recently Used (LRU) Cache using Java.

## 🧠 Concepts Used
- HashMap
- Doubly Linked List
- Data Structures & Algorithms

## ⚙️ Features
- O(1) time complexity for get() and put()
- Efficient memory management
- Interactive console-based simulation

## 🔄 How It Works
- Most recently used items are moved to the front
- Least recently used items are removed when capacity is exceeded

## ▶️ How to Run

1. Compile:
   javac src/*.java

2. Run:
   java src.Main

## 📊 Example

put(1,10)
put(2,20)
get(1)
put(3,30) → removes key 2

## 📈 Time Complexity
- get(): O(1)
- put(): O(1)
